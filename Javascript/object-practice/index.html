Object Notes

  • Objects are collections of related properties and/or methods.
    This is useful since objects can be used to represent real world items or people and their various functions/ information.
    Like a person object would have a name property and a wave method in their object.
  • Objects use key:value to store an objects properties and function() to house the objects methods.
    Similar to how a class uses function() for their methods and how localStorage uses key:value to store their objects.
  • The basic way to create an object is through declaring a variable and then setting it equal to {}.
    Within the braces can be the various key:value and methods for the object.
  • Object properties can be accessed via appending'.' and the property name after the variable.
    For example, a person object with a firstName property can be accessed by typing person.firstName
  • Object properties can also include other objects or arrays. In this case the same practice can be used to access these properties as mentioned above.
    To access specific data from the object or array, appending '.' and the property name or '[]' and the items index respectively will allow you to access the desired data.
  • In addition to have initially declared objects, an object can be created as a constructor in the case of making multiple objects with similar properties or methods.
    Rather than declaring multiple objects using "const objectName = {}" you would use "function objectConstructorName(parameters) {}" and with this any newly declared variable that is set equal to the object constructor will inherit all of the predefined properties and methods, and any parameters passed can initially be set equal to those defined in the constructor.
    Example, a person object constructor can take in a firstName and lastName parameter and set them equal to the object's firstName and lastName properties upon the objects creation.
  • The delete keyword can be used to delete certain properties or methods from an object.
    Example, "delete Person.firstName" would remove the first name property from the person object.
  • To view all of the properties an object has, you can use the Object.getOwnPropertyNames() and pass in said object as a parameter.
    This is useful when iterating through all keys an object has and using it in conjunction with Object.values() and passing in the object again, will allow you to grab the property values to display those too.
