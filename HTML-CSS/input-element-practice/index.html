Tag Element Notes & Examples:

  • The input tag can be used to gather different types of info from the user, like numbers, text, or dates.
  • All input tags are made the same way, < input >, and to change the kind of input they accept, you would assign them a type="".
    Like radio, check, text, or password.
  • If you are using radio inputs, providing a name="", field and providing an identical name for all of the radio buttons will allow the user to only select a single one.
  • Labels can be utilized to provide text descriptions for checkboxes, radio buttons, and text boxes while also allowing the user to click on the label to access said input field.
  • For labels, adding the field, for="", and setting it equal to the id of a checkbox or radio box allows the user to click on the label and have it check a checkbox or select a radio button.
